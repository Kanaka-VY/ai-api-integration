# AI API Integration Assignment

This project demonstrates how to integrate multiple Generative AI APIs using Python in a structured and practical way.

## APIs Used

* OpenAI API
* Gemini API (Google)
* Cohere API
* Groq API
* HuggingFace API
* Ollama (Local LLM)

## Project Structure

```
ai-api-integration/
├── ai_chat.py
├── openai_example.py
├── gemini_example.py
├── cohere_example.py
├── groq_example.py
├── huggingface_example.py
├── ollama_example.py
├── requirements.txt
└── README.md
```

---

## Requirements

* Python 3.8 or higher
* Internet connection for cloud APIs
* Ollama installed (for local model)

---

## Installation

```bash
pip install -r requirements.txt
```

---

## How to Run

```bash
python openai_example.py
python gemini_example.py
python cohere_example.py
python groq_example.py
python huggingface_example.py
python ollama_example.py
```

For chatbot:

```bash
python ai_chat.py
```

---

## API Keys Setup

Set your API keys as environment variables:

* OPENAI_API_KEY
* GOOGLE_API_KEY
* COHERE_API_KEY
* GROQ_API_KEY
* HUGGINGFACE_API_KEY
